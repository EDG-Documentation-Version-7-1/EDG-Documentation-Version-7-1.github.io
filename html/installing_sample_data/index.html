Installing Sample DataÔÉÅ

TopQuadrant provides sample data to use for training and getting starting in TopBraid EDG. Below are instructions on how to install the sample data in your environment.

  1. Download samples from https://www.topquadrant.com/topbraid-composer-install/ for EDG 7.2

  2. Click the plus button in the top right of EDG home page. Select the zip file you downloaded.
