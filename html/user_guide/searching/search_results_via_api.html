Search results via API

Search results and facets can be accessed via API. Results are returned as JSON.

Service Syntax

http://…/search/results

http://…/search/facets

Arguments

term

search term or phrase (uses query syntax)

optional

limit

number of results returned

optional

offset

number for offset

optional

withFacets

true or false

optional
