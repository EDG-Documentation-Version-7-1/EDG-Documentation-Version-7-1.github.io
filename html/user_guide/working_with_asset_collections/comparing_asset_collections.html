Comparing Asset Collections

Comparison Report is available under the Reports tab. When used while in the context of a production copy of a collection, it compares the production copy of the current collection with the production copy of another selected collection. When used while in the context of a working copy, it compares the working copy and production copy of the current collection.

This report only compares information contained directly in the selected collection. It does not compare contents of included asset collections. The report lists each changed asset and properties that were changed, showing the changed values. If a value was added, it is shown in a green background. If a value was deleted, it is shown in a pink background. Selecting the asset link opens the edit form for an asset in the report.
